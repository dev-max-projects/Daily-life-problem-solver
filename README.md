# 🧰 Daily Life Problem Solvers Web App

A simple web application built using **HTML, CSS, and JavaScript** that helps solve small daily problems with useful tools.

## Features
- 💰 Tip Calculator – Calculate tip based on bill amount and percentage
- 💱 Currency Converter – Convert currency values
- 📝 To-Do List – Add and manage daily tasks

## Technologies Used
- HTML
- CSS
- JavaScript

## Tools in the App
1. Tip Calculator  
2. Currency Converter  
3. To-Do List

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
